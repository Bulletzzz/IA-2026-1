# Fontes dos dados

Tema: análise de treino e exercícios de academia.

Os datasets não são versionados (ver `.gitignore`). Para baixar, configure o token
do Kaggle em `~/.kaggle/access_token` e rode `python baixar_dados.py`.

## Tabular — problemas clássicos (ML + DL)
- **Nome:** Gym Members Exercise Dataset
- **Autor:** valakhorasani
- **Link:** https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset
- **Licença:** Apache 2.0
- **Tamanho:** 973 registros, 15 colunas
- **Alvo escolhido:** `Experience_Level` (3 classes: iniciante, intermediário, avançado)
- **Atributos:** idade, gênero, peso, altura, BPM máx/médio/repouso, duração da sessão,
  calorias queimadas, tipo de treino, % de gordura, ingestão de água, frequência semanal, IMC

## Imagens — visão computacional (pré-treinado + treinado)
- **Nome:** Workout/Exercise Images
- **Autor:** hasyimabdillah
- **Link:** https://www.kaggle.com/datasets/hasyimabdillah/workoutexercises-images
- **Licença:** CC BY-NC-SA 4.0
- **Tamanho:** 13.853 imagens, 22 classes
- **Classes:** barbell biceps curl, bench press, chest fly machine, deadlift,
  decline bench press, hammer curl, hip thrust, incline bench press, lat pulldown,
  lateral raises, leg extension, leg raises, plank, pull up, push up, romanian deadlift,
  russian twist, shoulder press, squat, t bar row, tricep dips, tricep pushdown
